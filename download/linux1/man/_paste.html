past
 сцепление строк файлов  

СИНТАКСИС
  
paste [опции] файл1 [файл2...]

ОПИСАНИЕ
 

Команда paste выводит в стандартный вывод строки, состоящие из последовательного сцепления соответствующих строк каждого файла. Если не указано ни одного файла или задано имя -, читается стандартный ввод.


ОПЦИИ
 
-d строка ,--delimiters=СПИСОК если эта опция не указывается, то между соединяемыми строками вставляется символ табуляции; если опция указана, то вместо табуляции используются символы из строки (списка) (строка используется как кольцевой список).Вы можете указать одиночный символ (- d ":"), который будет использовать везде, либо список символов (-d "xyz"), которые будут использоваться последовательно в строке (первый разделитель - х, затем - у, затем - z, затем - х, затем - у,...). Аргументы, обязательные для длинных ключей, обязательны и для коротких.
-s , --serial выполняется сцепление не единичных строк одновременно из всех входных файлов, а сцепление последовательных строк файла1, затем файла2 и т.д. (обрабатывать файлы последовательно). Переставить местами строки и колонки при выводе.

СМ.ТАКЖЕ
  cut, join.
