unset
  удаляет переменную из окружения   


СИНТАКСИС
  
 
    unset [-fv] name ...

ОПИСАНИЕ
  Команда set удаляет переменную из окружения, фактически -- устанавливает ее значение в null. Обратите внимание: эта команда не может сбрасывать позиционные параметры (аргументы).

ОПЦИИ 

Опции см man unset.
