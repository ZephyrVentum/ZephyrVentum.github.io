w
   выводит текущий процесс для каждого работающего в системе пользователя.   

Синтаксис
  
         w [опции] [пользователь]

Описание
 

Команда w выводит текущий процесс для каждого работающего в системе пользователя, или, более точно, для каждого командного процессора каждого пользователя.

Верхняя строка вывода аналогична строке, которую выводит команда uptime. Далее выводятся следующие колонки 

USER     имя пользователя
TTY      имя терминала или удаленного хоста  
LOGIN    время входа в систему
IDLE     время простоя
JCPU     это процессорное время, использованное всеми процессами терминала. Оно не включает 
	 фоновые задания, запускаемые ранее, однако включает, фоновые задания, 
	 работающие в текущее время.
PCPU 	 это процессорное время,  использованное текущим процессом, указанным в поле "WHAT"
WHAT	 командная строка текущего процесса

Укажите имя пользователя, чтобы
увидеть информацию только об этом пользователе.

ОПЦИИ 

    -V			Версия программы
    -h			He выводить строку заголовка
    -f			He выводить колонку FROM
    -s			He выводить колонки JCPU и PCPU
    -u			Игнорировать имя пользователя, если фигурирует текущий
			процесс и процессорное время.
