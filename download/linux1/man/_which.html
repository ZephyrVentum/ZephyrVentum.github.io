which
   поиск исполняемых файлов в пути поиска командного процессора   

Синтаксис
  
 which [options] [--] programname [...]

Описание
 

Команда which вернет полный путь к programname . Если несколько программ в пути поиска имеют одинаковые названия (скажем, /usr/bin/who и /usr/local/ bin/who), то команда which вьщаст первую из них, а выполнение команды с опцией -а позволит получить все пути.
