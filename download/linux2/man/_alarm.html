alarm (2)
   посылка сигнала - будильника текущему процессу  

Синтаксис
  
    #include <signal.h>

    unsigned int alarm(unsigned int sec);

Описание
  alarm посылает текущему процессу сигнал SIGALRM по истечении заданного интервала времени.

Параметры
  sec - временной интервал в секундах.

Возвращаемое значение
  Временной интервал, установленный при предыдущем обращении к alarm.

См.также
  kill, raise, setitimer. signal.
