chmod, fchmod (2)
  изменение прав доступа к файлу  

Синтаксис
  
    #include <sys/types.h>
    #include <sys/stat.h>

    int chmod(char *fname, mode_t mode);
    int fchmod(int fd, mode_t mode);

Описание
  Изменяет права доступа к файлу, заданному именем fname (chmod) или файловым дескриптором fd (chmod). Режимы задаются параметром mask как результат логической операции OR к следующим константам:
S_ISUID    040008
 при выполнении установить идентификатор пользователя
S_ISGID    020008
 при выполнении установить идентификатор группы
S_IRUSR (S_IREAD)    004008
 владелец может читать
S_IWUSR (S_IWRITE)    002008
 владелец может писать
S_IXUSR (S_IEXEC)    001008
 владелец может выполнять файл или искать в каталоге
S_IRGRP    000408
 группа-владелец может читать
S_IWGRP    000208
 группа-владелец может писать
S_IXGRP    000108
 группа-владелец может выполнять файл или искать в каталоге
S_IROTH    000048
 все остальные могут читать
S_IWOTH    000028
 все остальные могут писать
S_IXOTH    000018
 все остальные могут выполнять файл или искать в каталоге

Возвращаемое значение
  При нормальном выполнении chmod возвращает 0. При ошибках выполнения возвращает -1 и устанавливает errno.
