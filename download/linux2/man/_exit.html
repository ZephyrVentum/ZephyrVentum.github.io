exit (3)
   нормальное завершение процесса   

Синтаксис
  
    #include <unistd.h>

    void exit(int status);

Описание
 

exit немедленно завершает текущий процесс.

При завершении закрываются все открытые процессом файлы, все дочерние процессы завершившегося процесса становятся наследниками процесса 1 (процесса init),

Параметры
 

Параметр status определяет, код завершения процесса, этот код может быть прочитан родительским процессом в системном вызове wait.
