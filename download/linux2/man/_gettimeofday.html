gettimeofday (2)
   получение времени дня  

Синтаксис
  
       
    #include <sys/time.h>
    #include <unistd.h>

    int gettimeofday(struct timeval *tm, struct timezone *tz);

Описание
  gettimeof записывает время дня в структуру, на которую указывает аргумент tm. Аргумент tz может задаваться равным NULL. Описание структуры timeval следующее: 
    struct timeval {
           long tv_sec;        /*  секунды */
           long tv_usec;  /* микросекунды */
       };

Возвращаемое значение
  0 - при успешном выполнениие, -1 - при ошибке, в последнем случае устанавливается errno.

См.также
  asctime, ctime, gmtime, mktime, strftime, time.
