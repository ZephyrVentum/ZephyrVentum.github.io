perror (3)
   получение сообщение об ошибке  

Синтаксис
  
    #include <stdio.h>

    void perror(const char *s);

    #include <errno.h>

    const char *sys_errlist[];
    int sys_nerr;

Описание
 

Функция perror заносит в символьную строку, указатель на которую является аргументом функции s, диагностическое сообщение об ошибке, код которой задается значением переменной errno.

Чтобы получить текст диагностического сообщения без перевода строки, может быть использован глобальный массив - список ошибок sys_errlist. Переменная sys_nerr содержит размер этого массива. Переменная errno может использоваться как индекс в этом массиве.
