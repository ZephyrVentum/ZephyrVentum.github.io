pipe (2)
   создание неименованного программного канала  

Синтаксис
  
    int pipe(int fd[2]);

Описание
  pipe создает неименованный программный канал и возвращает два файловых дескриптора: fd[0] - для чтения из канала и fd[1] - для заниси в канал.

Дальнейшая работа с каналом происходит так же, как с открытым файлом.

Возвращаемое значение
  При нормальном выполнении возвращает 0. При ошибках выполнения возвращает -1 и устанавливает errno.
