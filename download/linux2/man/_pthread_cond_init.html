phtread_cond_init (3)
   инициализация сигнального семафора  

Синтаксис
  
    # include <pthread.h>

    int pthread_cond_init(pthread_cond_t *cond, pthread_condattr_t *attr);

Описание
 

Функция pthread_cond_init инициализирует сигнальный семафор, на идентификатор которого указывает аргумент cond. Аргумент attr указывает на объект атрибутов сигнального семафора. При использовании стандартных атрибутов его значение может задаваться NULL.

Возвращаемое значение
  При успешном завершении pthread_cond_init возвращает 0. В случае ошибки возвращается ненулевое значение.

См.также
  pthread_cond_broadcast, pthread_cond_signal, pthread_cond_wait.
