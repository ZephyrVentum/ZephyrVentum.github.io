rand, srand (3)
   генератор случайных чисел   

Синтаксис
  
       
    #include <stdlib.h>

    int rand(void);

    void srand(unsigned int seed);

Описание
 

rand возвращает псевдослучайное число из интервала 0 - RAND_MAX.

srand инициализирует генератор случайных чисел. Параметр seed - начальное случайное число, в этом качестве можно использовать результат функции time.

Если Вы хотите получить случайное (псевдослучайное) число, подчиняющееся равномерному закону распределения на интервале, например от 1 до I, то следует употреблять функцию rand примерно в таком контексте: 

   m=1+(int) ((double)I*rand()/(RAND_MAX+1.0));
