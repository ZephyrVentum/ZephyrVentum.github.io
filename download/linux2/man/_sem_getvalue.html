sem_getvalue (3)
   получение значения общего семафора  

Синтаксис
  
    # include <pthread.h>

    int sem_getvalue(sem_t *sem, int *value);

Описание
 

Функция sem_getvalue записывает текущее значение общего семафора, на идентификатор которого указывает аргумент sem по адресу, задаваемому аргументом value.

Возвращаемое значение
  При успешном завершении sem_getvalue возвращает 0. В случае ошибки возвращается ненулевое значение.

См.также
  sem_init, sem_post, sem_wait.
