semget (2)
   получение идентификатора массива семафоров  

Синтаксис
  
    # include <sys/types.h>
    # include <sys/ipc.h>
    # include <sys/sem.h>

    int semget ( key_t key, int nsems, int semflg )

Описание
  semget возвращает идентификатор массива из nsem семафоров, связанного с ключом, значение которого задано аргументом key. Если массива семафоров, связанного с таким ключом, нет и в параметре semflg имеется значение IPC_CREATE или значение ключа задано IPC_PRIVATE, создается новый массив семафоров. Значение ключа IPC_PRIVATE гарантирует уникальность идентификации нового массива семафоров.

Значение параметра semflg формируется как логическое ИЛИ одного из значений: IPC_CREAT (создать новый массив семафоров) или IPC_EXCL (получить идентификатор существующего массива) и 9 бит прав доступа (см. chmod).

Возвращаемое значение
  При успешном завершении возвращается неотрицательное целое число - идентификатор множества семафоров. В случае ошибки возвращается -1 и устанавливается код ошибки в errno.
