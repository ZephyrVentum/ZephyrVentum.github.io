Диалоговое окно. Confirm.
Confirm является таким же диалоговым окном как Alert, но в отличии от него он имеет две кнопки, Ок и Отмена. 

if (window.confirm("Нажмите одну из кнопок.")) {
window.alert("Нажата кнопка ОК.");
}
else {
window.alert("Нажата кнопка Cancel. ");
}

Заменяйте код после else если хотите увидеть другой результат. 

if (window.confirm("Нажмите одну из кнопок.")) {
window.alert("Нажата кнопка ОК.");
document.write("Нажата кнопка Ок");
}
else {
window.alert("Нажата кнопка Cancel. ");
document.write("Нажата кнопка Отмена");
}
