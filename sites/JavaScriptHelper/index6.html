Функции

Функция это фрагмент кода JavaScript, который можно вызвать из любого места в программе. Функция описываеться с помощью ключевого слова function. 

function <Имя функции> ([<Параметры>])
{
<Тело функции>
[return <Значение>]
}
			
function f_Alert_OK()
{
window.alert("Успешное выполнение функции");
}
			
function f_Alert_(msg)
{
window.alert(msg);
}
