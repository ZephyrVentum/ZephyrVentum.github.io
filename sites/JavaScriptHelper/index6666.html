Пример использования функций

Приведем пример использования функций на практике а так же их вызов с помощью тега input. 

<body>
      <input type="button" value="fun1" onclick="f_Sum('button1')"></input>
      <input type="button" value="fun2" onclick="f_Sum('button2')"></input>
</body>
			
			
<script type="text/javascript">
	function f_Sum(id)
	{
			if (id == "button1")
		{
			document.write('Ок');
		}
			else 
		{
			document.write('Отмена');
		}
	}
</script>
